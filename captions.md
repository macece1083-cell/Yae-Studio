# Yae Studio — Caption Şablonları

Her post paylaşmadan önce bu şablonlardan birini al, [ ] içindeki kısımları doldur.

---

## Tip 1 — Günlük Kombin

```
[renk/tema] bir [mevsim/gün] kombini ✦

Basit ama etkili. Bu seferki favorimiz [öne çıkan parça].

Tüm ürünler → bio linkinde 🔗
```

**Örnek:**
```
Krem tonlarında bir Pazartesi kombini ✦

Basit ama etkili. Bu seferki favorimiz oversize blazer.

Tüm ürünler → bio linkinde 🔗
```

---

## Tip 2 — Ürün Odaklı Post

```
Tek parça, sonsuz kombinasyon.

[ürün adı] — hem [durum 1] hem [durum 2] için.

Link bio'da 👆
```

**Örnek:**
```
Tek parça, sonsuz kombinasyon.

Kahverengi trençkot — hem ofis hem akşam yemeği için.

Link bio'da 👆
```

---

## Tip 3 — İlham Verici

```
Stili karmaşıklaştırmaya gerek yok.

[basit kural veya fikir]

Kombin detayları → bio 🔗
```

**Örnek:**
```
Stili karmaşıklaştırmaya gerek yok.

İki nötr renk + bir statement parça. Hepsi bu.

Kombin detayları → bio 🔗
```

---

## Tip 4 — Mevsim Geçişi

```
[mevsim]'den [mevsim]'e geçerken dolabınızda olması gereken [sayı] parça:

1. [parça]
2. [parça]
3. [parça]

Hepsi Trendyol/Milla'dan → bio linkinde bulabilirsiniz 🔗
```

---

## Tip 5 — Soru ile Etkileşim

```
[fiyat aralığı] altında kombin yapılır mı?

Biz yaptık. [tema] temalı bu kombin toplamda [fiyat] ✦

Parçalar bio linkinde 🔗

Siz hangi fiyat aralığında alışveriş yapıyorsunuz? 👇
```
